## 🚀 Getting Started Locally

### Prerequisites

- Node.js and npm installed

### Setup Instructions

```bash
# Clone the repository
git clone https://github.com/abhi-gadge1773/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm start
